# AstroFusion

A visually engaging space-themed website built with HTML, CSS & JavaScript.  
Designed to showcase content related to space, exploration, missions, rockets and more, with immersive backgrounds and interactive elements.

## Demo
You can run the site locally by opening `index.html` in your browser, or deploy it via a static site host (e.g., Render, Netlify or GitHub Pages).

## Features
- Multiple HTML pages themed around space (e.g., “Mission Impossible”, “Galactic Colonies”, “RocketMan”)  
- Background videos and high-resolution imagery for a cinematic effect  
- Responsive design with CSS styles for various pages and layouts  
- Interactive navigation via `main.js` and `script.js` for dynamic behavior  
- Rich media content including `.mp4` backgrounds and optimized images  

## Tech Stack
- HTML5  
- CSS3 (multiple style sheets: `style.css`, `styles.css`, etc.)  
- JavaScript for interactivity  
- Static assets: video files (`bg.mp4`), images (`*.jpg`, `*.png`), icons
